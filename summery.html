React Js Brief

  • React's goal is to render HTML in a web page.
  • React renders HTML to the web page by using a function called createRoot() and its method render().
  • purpose of the function ( createRoot() ) is to define the HTML element where a React component should be displayed
  • render() method is then called to define the React component that should be rendered

JSX

  • JSX is not compulsary.
  • JSX stands for JavaScript XML.
  • JSX allows us to write HTML in React.
  • JSX makes it easier to write and add HTML in React.
  • JSX allows us to write HTML elements without any createElement() and/or appendChild() methods.
  • JSX converts HTML tags into react elements.
  • React supports if statements, but not inside JSX.
  • in order to embed a JavaScript expression inside JSX, the JavaScript must be wrapped with curly braces, {}.

Component

Components are like functions that return HTML elements. two types, Class components and Function components,

Prop

Props are arguments passed into React components. React Props are read-only! You will get an error if you try to change their value.

Event

Just like HTML DOM events, React can perform actions based on user events. React has the same events as HTML: click, change, mouseover etc.
Adding Events React events are written in camelCase syntax: onClick instead of onclick. React event handlers are written inside curly braces: onClick={shoot} instead of onclick="shoot()".
